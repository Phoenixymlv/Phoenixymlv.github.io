Custom Stack Collection

Four opinionated stacks, each with a clear identity and purpose.

This page presents four custom‑designed stacks, each built around a distinct philosophy of modern web development. Instead of forcing one “universal” setup, these stacks embrace the idea that different problems deserve different tools. Use this page as a reference after landing on the home page, to decide which stack best fits the project you’re about to build.

Feather Stack

LMHV Stack

Laravel • MySQL • htmx • Vite

Identity: A lightweight, server‑driven stack with minimal JavaScript and fast iteration.

Best for

  • CRUD apps and admin dashboards
  • Internal tools and back‑office interfaces
  • Projects where SEO, speed, and simplicity matter

Why it works

  • Laravel handles routing, auth, and data logic
  • htmx adds interactivity via HTML attributes instead of heavy SPA frameworks
  • Vite provides fast asset bundling; MySQL offers a stable relational core
Nuxt SSR Stack

NPHS Stack

Nuxt • PostgreSQL/Prisma • PostCSS • HTML • Sass

Identity: A Vue‑powered SSR stack for content‑rich, design‑driven applications.

Best for

  • SEO‑heavy sites, blogs, portfolios, marketing pages
  • Hybrid SSR/SSG experiences
  • Projects where layout and visual polish matter

Why it works

  • Nuxt delivers SSR/SSG for performance and SEO
  • Prisma + PostgreSQL provide a modern, type‑safe backend
  • Sass + PostCSS form a powerful, maintainable CSS pipeline
TypeScript Full‑Stack

VPTNV Stack

Vue • Prisma • TypeScript • Node • Vite

Identity: A clean, fully type‑safe full‑stack architecture built around TypeScript.

Best for

  • SaaS products and dashboards
  • Complex SPAs and data‑driven interfaces
  • Teams that value type safety and long‑term maintainability

Why it works

  • TypeScript runs through backend and frontend for consistent types
  • Prisma offers type‑safe database access on top of PostgreSQL
  • Vue + Vite provide a fast, modern frontend workflow
Maximal TS Ecosystem

3T4VPeX Stack

TypeScript • tRPC • Tailwind • Vue • Vite • Vitest • Prisma • Express

Identity: A feature‑rich, TypeScript‑centric stack for ambitious, complex applications.

Best for

  • Large‑scale apps with complex frontends
  • Real‑time or highly interactive interfaces
  • Teams that want deep control over both backend and frontend

Why it works

  • tRPC enables end‑to‑end type‑safe communication
  • Tailwind accelerates UI building with utility‑first CSS
  • Express + Prisma + PostgreSQL form a flexible, powerful backend
  • Vite + Vitest provide modern tooling and testing for Vue + TS
